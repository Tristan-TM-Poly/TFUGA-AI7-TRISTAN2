# OAKShield Core

OAKShield Core is the defensive, permissioned governance nucleus for this repository. It converts repository review into scope, sanitized observations, remediation plans, regression checks, and M-minus memory.

This pack intentionally avoids workflow, script, and configuration changes. It is documentation-first and safe to review before any automation is added.

## OAK boundary

Allowed:

- repository documentation review;
- local file and structure review;
- sanitized observations;
- remediation planning;
- regression planning;
- M-minus memory.

Rejected:

- work outside scope;
- third-party targeting;
- sensitive data collection;
- disruptive behavior;
- hidden persistence;
- unsafe instructions;
- bypass of protections.

## Core loop

```text
authorization
→ repository hygiene review
→ sanitized observation
→ fix plan
→ regression check
→ M-minus case
→ OAK verdict
```
