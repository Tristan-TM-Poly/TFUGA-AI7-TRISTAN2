# OAKShield Claim Style: ARK-SP-CUBE-GAIA v0.9

Status: C_style_guardrail

This guide keeps ARK-M1, SP-CUBE and GAIA-SAT language public-safe and review-safe.

## Core rule

Use the strongest wording justified by the weakest evidence gate.

```text
idea < formalized < coded < tested < simulated < prototyped < measured < externally reviewed < certified
```

## Allowed replacements

| Risky wording | Replace with |
|---|---|
| guaranteed revenue | simulated impact value, not revenue |
| certified climate credit | MRV-dependent climate-impact scenario |
| energy from vacuum | radiative heat transfer to colder environment |
| proven reactor | low-power energy/thermal architecture |
| official infrastructure finding | demo analysis requiring qualified review |
| patent-proven invention | patent-like technical record or claim tree |
| AI decision | review-support signal |
| automatic public-sector action | local draft requiring human review |

## Module style

### ARK-M1

Allowed: conceptual bench, low-power thermal architecture, prototype path.

Blocked: reactor, certified device, energy generator, proven efficiency claim without measurement.

### SP-CUBE

Allowed: radiative-cooling hypothesis, spectral separation model, IR-emission design candidate.

Blocked: energy from vacuum, guaranteed cooling, certified metamaterial, transparent blackbody in the same passive band/state.

### GAIA-SAT

Allowed: simulated gross climate-impact value, MRV-adjusted scenario, OAK-adjusted scenario.

Blocked: revenue, investment return, certified credits, enterprise valuation.

### Methane MRV

Allowed: simulated plume, candidate repair priority, MRV pipeline hypothesis.

Blocked: official detection, verified repair, certified reduction without evidence.

## Required suffixes

For simulated values, append:

```text
simulated / MRV-dependent / not revenue / not investment certified
```

For prototypes, append:

```text
prototype path / not certified / human review required
```

For public-sector or infrastructure outputs, append:

```text
review-support only / not a final decision
```

## Safe canonical sentence

ARK-SP-CUBE-GAIA is a research scaffold for turning thermal and climate ideas into structured claims, simulations, local tests, proof ledgers and OAK-safe review routes.
