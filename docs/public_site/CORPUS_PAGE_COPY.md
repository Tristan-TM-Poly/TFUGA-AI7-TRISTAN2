## Core systems

### OAK

```text
Preuve, falsification, robustesse, mémoire négative.
Une idée n'est pas une vérité tant qu'elle n'a pas traversé le test.
```

### HGFM

```text
Hypergraphes fractals mycéliens : représenter des systèmes comme réseaux vivants,
multi-échelles, connectés et évolutifs.
```

### CVCD

```text
Compression fertile d'invariants : extraire ce qui survit au bruit,
ce qui se répète, ce qui peut devenir théorie, prototype ou produit.
```

### Bayes-Tristan

```text
Décision probabiliste augmentée : distinguer vérité, utilité,
fertilité, risque, testabilité et action suivante.
```

### SAGE / AIT

```text
Agents spécialisés de recherche, synthèse, action, preuve, produit et compagnie.
```

### ZERO-TOUCH

```text
Automatisation maximale, friction minimale : transformer l'idée en artefact,
code, rapport ou PR sans perte d'énergie inutile.
```

### JKD-Tristan

```text
Simplicité, interception, adaptation, efficacité : couper le faux mouvement,
garder ce qui agit, rejeter ce qui surcharge.
```
