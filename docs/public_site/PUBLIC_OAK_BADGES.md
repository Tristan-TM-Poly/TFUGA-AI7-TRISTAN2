# Public OAK Badges

Status: B — public claim labeling system
Date: 2026-07-06

## Purpose

Public OAK Badges prevent overclaiming. They tell visitors what each page, system or product actually is.

## Badge set

| Badge | Meaning | Not allowed to imply |
|---|---|---|
| Vision | Direction, mission, imagination | proof, product, revenue |
| Doctrine | structured concept or framework | execution or validation |
| Prototype | concrete artifact or code exists | reliability or market demand |
| Tested | tests or reproducible checks exist | full correctness |
| Merged | integrated into main branch | complete product |
| Product Candidate | offer can be packaged | validated demand |
| Commercial Hypothesis | price/offer under exploration | guaranteed revenue |
| Validated Offer | real demand or sale exists | unlimited scale |
| Company Path | venture direction exists | incorporated/operating company |
