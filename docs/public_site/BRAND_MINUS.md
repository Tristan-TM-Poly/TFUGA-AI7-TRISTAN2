# Brand M-minus — Public Site

Status: B — public brand negative memory
Date: 2026-07-06

## Purpose

Brand M-minus records public-identity errors before they become reputational debt.

```text
A strong brand needs limits.
A public myth needs proof.
A symbol needs responsibility.
```

## Error classes

| Error | Risk | Countermeasure |
|---|---|---|
| too mystical | unclear, unverifiable | add proof ladder and concrete artifacts |
| too arrogant | reduces trust | add limits, humility, tests |
| too vague | no conversion | add product, output, next action |
| too technical | unreadable | compress into JKD public language |
| culturally risky | trust and respect failure | use Cultural Respect Gate |
| too commercial | feels extractive | add proof, value, limits, reciprocity |
| too personal | privacy/reputation risk | move details to manifesto or omit |
| too grandiose | looks ungrounded | add status labels and evidence |
| unverifiable | weak trust | link to proof ledger |

## Rewrite patterns

### Overclaim

```text
Bad: I build systems that transform governments.
Better: I prototype OAK-safe public-data systems that help map sources, risks and reports.
```

### Revenue overclaim

```text
Bad: This product will generate revenue.
Better: This is a commercial hypothesis that needs a paid pilot.
```

### Proof overclaim

```text
Bad: This theory is proven.
Better: This branch is a doctrine/prototype until tested and measured.
```

### Culture overclaim

```text
Bad: I represent this living tradition.
Better: I use this as a personal responsibility symbol and do not claim representation.
```

## M-minus seeds

```yaml
- error_type: too_mystical
  fix: add proof and artifact
- error_type: too_grandiose
  fix: add status badge and limit
- error_type: too_commercial
  fix: add OAK commercial disclaimer
- error_type: cultural_context_missing
  fix: remove or add respectful context
- error_type: product_without_validation
  fix: mark as commercial hypothesis
```

## Final rule

```text
If the claim becomes larger than the proof, shrink the claim or grow the proof.
```
