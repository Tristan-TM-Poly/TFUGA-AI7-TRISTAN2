## Safety posture

The site must stay OAK-safe:

- no claim of representing First Nations or communities ;
- no cultural reference used as decoration ;
- no market or revenue guarantee ;
- no proof claim without testing language ;
- no health, legal, justice or government automation claim without human authority ;
- no intrusive collection of personal data.
