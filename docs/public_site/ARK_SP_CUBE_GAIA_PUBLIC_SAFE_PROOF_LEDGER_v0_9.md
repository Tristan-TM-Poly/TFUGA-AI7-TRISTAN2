# ARK-SP-CUBE-GAIA Public-Safe Proof Ledger v0.9

Status: C_public_safe_ledger_scaffold

This ledger translates ARK-M1, SP-CUBE and GAIA-SAT claims into public-safe proof levels. It is a communication guardrail, not a certification record.

## Proof levels

| Level | Meaning | Allowed public wording |
|---|---|---|
| IDEA | Concept named and bounded | concept, hypothesis, research direction |
| FORMALIZED | Equations/docs exist | formalized model or architecture |
| CODED | Code exists | prototype code or local model |
| TESTED_LOCAL | Local tests pass | locally tested scaffold |
| SIMULATED | Synthetic/model outputs exist | simulated scenario |
| PROTOTYPED | Physical or UI prototype exists | prototype, not certified |
| MEASURED | Real measurement exists | measured in stated conditions |
| EXTERNAL_REVIEW | Third-party review exists | reviewed, not necessarily certified |
| CERTIFIED | Recognized external certification | certified under named standard |
| BLOCKED | Unsafe or overclaimed | do not publish as claim |
