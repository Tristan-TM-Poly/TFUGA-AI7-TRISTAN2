1. **Feu d'origine** — mémoire, racines, naissance à Laval, Québec, Canada, Gaia.
2. **Feu de mémoire** — M-, réparation, vérité, sources honorées.
3. **Feu de fracture** — ce qui doit être corrigé dans les systèmes.
4. **Feu de preuve** — OAK, test, falsification, robustesse.
5. **Feu de réparation** — justice, Gaia, responsabilité.
6. **Feu de co-création** — rien sur les communautés sans les communautés.
7. **Feu de transmission** — systèmes vivants, compagnies, infrastructures.
8. **Feu du système vivant** — technologie qui protège au lieu d'extraire.
