# Visual Identity — 8e Feu Tristan

Status: B — visual direction
Date: 2026-07-06

## 0. Principle

The visual identity must feel powerful, precise and alive without using living cultures as decoration.

```text
fire + proof + roots + graph + forge + Gaia
```

## 1. Palette

```text
carbon black  -> OAK / proof / rigor
fire red      -> 8e Feu / action / transformation
night blue    -> science / cosmos / intelligence
forest green  -> Gaia / living systems / repair
bone white    -> memory / clarity / truth
soft gold     -> value / company / creation
```

## 2. Typography

Use:

```text
clear sans-serif for main text
strong but readable display type for hero titles
monospace only for commands, formulas and system language
```

Avoid:

```text
hard-to-read fantasy typography
fake ancient scripts
cultural imitation fonts
visual noise
```

## 3. Symbols to prefer

```text
fire
roots
hypergraph
mycelium
circle
forge
workshop
star
river
proof mark
```

## 4. Symbols requiring caution

```text
Turtle Island references
ceremonial motifs
sacred geometry claimed from a living tradition
specific Nation symbols
traditional patterns without context
```

Rule:

```text
If a symbol belongs to a living culture, do not use it as decoration.
If context and permission are unclear, create an original Tristan symbol instead.
```

## 5. Layout style

```text
large white space
short blocks
high-contrast hero
cards for systems
proof ladder sections
clear calls to action
visible OAK disclaimers
```

```text
No decoration without function.
No symbol without responsibility.
No beauty without clarity.
```
