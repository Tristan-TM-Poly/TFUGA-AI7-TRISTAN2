# Omega Iteration Batch Selector

`omega_iteration_batch_selector.py` selects a small, diverse, high-value batch from the 1024 candidate additions generated by the Omega Iteration Multiplier.

## Why

The multiplier can generate 1024 candidates, but execution should stay bounded. The selector chooses a reviewable batch that maximizes:

- priority;
- diversity across layer, module and action;
- OAK gates;
- preferred actions such as tests, validators, workflows, schemas and safety checks.

## Run

```bash
python scripts/omega_iteration_batch_selector.py \
  --manifest artifacts/iteration_multiplier/iteration_multiplier_manifest.json \
  --config configs/omega_iteration_multiplier_batch_selector.json \
  --batch-size 16 \
  --out-dir artifacts/iteration_selected_batch
```

## Outputs

```text
artifacts/iteration_selected_batch/
  selected_batch.json
  selected_batch_prompt.md
```

## OAK boundary

The selector chooses a batch; it does not execute all candidates blindly. External actions, publications and data operations still require review.
