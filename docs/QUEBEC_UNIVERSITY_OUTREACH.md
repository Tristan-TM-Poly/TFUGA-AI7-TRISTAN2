# Quebec University Outreach — AIT-Tristan2 / Ω-DE-TensorProd∞ / Ω-ROI-OAK

This folder contains a draft outreach campaign for Quebec universities. It is designed for careful, targeted outreach only, not mass sending.

## Goal

```text
Open exploratory collaboration conversations with Quebec universities around scientific AI, nonlinear systems, research/patent digestion, energy, datacenters, batteries, materials, and OAK-safe commercialization.
```

| File | Role |

## Target waves

```text
Wave 1: ÉTS, Polytechnique Montréal, INRS, Université Laval, Université de Sherbrooke
Wave 2: UdeM, HEC, McGill, Concordia, UQTR, UQAC
Wave 3: UQAM, UQAR, UQO, UQAT, ENAP, TÉLUQ, Bishop's, BCI
```

## OAK-safe rule

```text
Do not send anything externally without verified contact, targeted adaptation, and explicit approval.
```

## Core positioning

```text
A prototype framework for converting complex scientific systems, publications, patents, code, benchmarks, and industrial constraints into verifiable reports, prototypes, and risk-adjusted collaboration opportunities.
```
