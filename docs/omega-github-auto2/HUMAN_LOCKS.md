# Omega AUTO2 Human Locks

AUTO2 is designed for zero-touch generation, not zero-review deployment.

## Always blocked without explicit human approval

- Merge to `main`
- External outreach or customer messaging
- Production billing changes
- Patentable disclosure
- Trade-secret disclosure
- Regulated-domain compliance claims
- Destructive repository operations
- Publication of customer or private data

## Allowed by default

- Generate manifests
- Generate draft issues
- Generate draft PR specs
- Generate Codex task packs
- Generate dashboards
- Run tests
- Compute E-cap scores
- Create OAK reports
- Record M-minus failure notes

## Principle

```text
No irreversible action from automation.
```

The system may recommend. It may draft. It may rank. It may test. It may not commit the ecosystem to legal, financial, IP, or customer-facing consequences without review.
