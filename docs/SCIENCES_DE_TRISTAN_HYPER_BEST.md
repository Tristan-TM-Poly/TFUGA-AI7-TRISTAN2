## 3. TheoryReactor

The TheoryReactor turns one idea into a research ecosystem.

```text
TheoryReactor(I) = {H_i, T_i, P_i, E_i, R_i}_{i=1..n}
```

Where:

- `H_i`: child hypotheses;
- `T_i`: candidate theory genomes;
- `P_i`: prototype contracts;
- `E_i`: experiment contracts;
- `R_i`: predicted residues and failure modes.

Pseudocode:

```python
def theory_reactor(raw_idea):
    safe_claim = ClaimTransmuter().transmute(raw_idea)
    split_claims = HypothesisSplitter().split(safe_claim)

    organisms = []
    for claim in split_claims:
        card = ScienceCard.from_claim(claim)
        organism = ScienceOrganism.from_card(card)
        organism.genome = TheoryGenome().build(card)
        organism.immune_system = MemoryMinusEngine().detect(card)
        organism.oak_review = OAKCourt().review(card)
        organism.prototype_contract = PrototypeForge().plan(card)
        organism.experiment_contract = ExperimentForge().plan(card)
        organisms.append(organism)

    return HGFM().insert_many(organisms)
```

Law:

```text
Tu ne développes plus une idée ; tu la fais fissionner en écosystème de recherches.
```

```yaml
    - F1
```

## 14. AutonomousResearchLoop

```python
def autonomous_research_loop(portfolio):
    cards = load_cards(portfolio)
    for card in cards:
        organism = ScienceOrganism.from_card(card)
        organism = ClaimTransmuter().apply(organism)
        organism = MemoryMinusEngine().apply(organism)
        organism = BayesTristanTensor().score(organism)
        organism = OAKCourt().review(organism)

        if organism.verdict == "split":
            children = HypothesisSplitter().split(organism)
            enqueue(children)

        if organism.ready_for_prototype:
            prototype = PrototypeForge().build_contract(organism)
            benchmark = BenchmarkArena().build_contract(prototype)

        if organism.has_result:
            residues = ResidueMiner().mine(organism.result)
            update_memory(residues)

        if organism.ready_for_paper:
            PaperForge().draft(organism)

        update_canon(organism)
```

OAK constraint:

```text

autonomy = auto-generation of next verifiable tests
```
