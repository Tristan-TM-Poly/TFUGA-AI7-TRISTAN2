# Omega Action Minimalism

This repository uses a safe, reversible, test-first workflow.

Core loop:

```text
observe -> choose a small step -> test -> record lesson -> improve
```

Rules:

- keep changes small;
- prefer reversible edits;
- test before promotion;
- record lessons from failed attempts;
- keep claims humble until verified.

Canonical implementation lives in Tristan-TM-Poly/TTM-TFUGA-AI7-TRISTAN2.
