## II. Chaine causale maitresse

The architecture is a thermodynamic/electrodynamic cascade where each physical effect activates the next:

1. **Gradient:** the network dissipates or routes heat, creating `DeltaT`.
2. **Signal:** mycelial junctions convert this gradient into voltage/current through a Seebeck-like stage.
3. **Field:** currents and circuits generate local static and multi-frequency `E` and `B` fields.
4. **Action:** these fields orient, separate, polarize, migrate, and concentrate molecules toward cavities.
5. **Transformation:** catalysts and adsorbents purify or transform the fluid.
6. **Proof:** differential Raman spectroscopy audits the before/after chemical state.
7. **Law:** CVCD compresses the run; OAK rejects illusions.

Mantra:

```text
The network cools, creates a gradient, generates a signal, applies fields,
transforms chemistry, then measures before and after.
```
