# QC-DIGEST Governance Docs

This folder contains governance files, output policy, milestone planning, and local workflow contracts for QC-DIGEST.

## Files

- `PUBLIC_RELEASE_CHECKLIST.md`: checks before public release.
- `OUTPUTS_POLICY.md`: rules for generated outputs, manifests, checksums, and review states.
- `V0_5_MILESTONE_PLAN.md`: v0.5 plan.
- `V0_5_TRACKER.md`: v0.5 tracker.
- `TRISTAN_LOCAL_WORKFLOW.md`: local workflow contract for Tristan-owned work.
