# Ω-UNI-QC-RESEARCH-ENGINE-T

## Mission

Ω-UNI-QC-RESEARCH-ENGINE-T is the Quebec university research operating system for Tristan. It turns universities, research axes, labs, professors, patents, publications, official contact routes, replies, microbenchmarks, GitHub assets, and OAK evidence into one executable pipeline.

```text
Toutes les universités québécoises
→ toutes leurs recherches
→ routes officielles
→ scoring OAK
→ messages adaptés
→ micro-benchmarks
→ rendez-vous
→ projets
→ preuves
→ partenariats
→ publications / brevets / revenus
```

## OAK constraints

- No guessed emails.
- No mass sending.
- No unsupported performance, ROI, partnership, or endorsement claims.
- Official source or no send.
- Form routes require confirmation records.
- One polite follow-up only unless invited to continue.
- Sensitive IP/public claims require review before disclosure.

## Canonical loop

```text
Discover → Score → Adapt → Send/Packet → Record → Monitor → Follow-up → Microbenchmark → OAK report → Partner/Publish/Patent/Revenue
```

## Command vocabulary

```text
GO UNI-SCAN          Discover official routes and research axes.
GO UNI-GREENIFY      Convert TO_DISCOVER/YELLOW to GREEN_EMAIL or GREEN_FORM.
GO UNI-MICROBENCH    Generate best microbenchmarks for a target.
GO UNI-SEND-GREEN    Send only approved GREEN_EMAIL routes.
GO UNI-FORM-PACKETS  Prepare official-form submissions.
GO UNI-REPLY         Route replies into next actions.
GO GITHUB-ALL-FORGE  Scan and improve all accessible GitHub repos.
GO OAK-REPORT        Produce a global OAK state report.
```
