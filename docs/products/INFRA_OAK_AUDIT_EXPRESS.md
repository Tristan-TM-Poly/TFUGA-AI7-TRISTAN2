# Infra-OAK Audit Express

Status: product candidate / commercial hypothesis
Date: 2026-07-06

## 0. Product sentence

```text
Infra-OAK Audit Express transforms an infrastructure claim, asset list or authorized document set into a public-safe evidence, risk, maintenance and resilience report.
```

## 1. Target users

```text
municipal teams
asset owners
infrastructure consultants
public-data / civic-tech teams
researchers
private operators with authorized data
founders exploring infrastructure products
```

## 2. Inputs

```text
asset list
public or authorized documents
known constraints
source notes
maintenance signals
risk concerns
what must not be published
```

## 3. Outputs

```text
source registry
asset graph
dependency graph
evidence items
risk tensors
maintenance priorities
resilience scenarios
OAK security gate result
Markdown report
JSON bundle
GraphML graph
next action map
```

## 5. Offer ladder

```text
Starter Audit — one small asset group / one claim / one report
Deep Audit — multiple assets, sources, risk tensors and scenarios
Graph Pack — JSON + GraphML + dashboard-ready bundle
Custom System — adapted InfraGraph model and report pipeline
```

## 6. Price hypotheses

```text
Starter: 149–299 EUR
Deep: 499–999 EUR
Graph Pack: 799–1500 EUR
Custom: scoped separately
```

OAK note: prices are commercial hypotheses until validated by real paid pilots.

## 7. What this product does not do

```text
It does not replace engineers, regulators, emergency managers or public authorities.
It does not publish sensitive details.
It does not guarantee savings, funding, repairs or official decisions.
It does not process real restricted data without authorization and review.
```

## 8. OAK delivery rule

```text
Show the sources.
Show the evidence.
Show the risks.
Show the limits.
Show the next test.
```
