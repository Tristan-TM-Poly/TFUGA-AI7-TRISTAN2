# Ω-SCI-PATENT-QC-DIGEST-T — Plus Ultra Architecture

## Core idea

The system is not only a literature review tool. It is a reusable Science↔IP intelligence kernel:

```text
sources → normalized documents → LOG/CVCD → OAK scores → HGFM graph → bridges → opportunities → release gates → reusable products
```

## Reusable layers

1. **Ingestion layer**: OpenAlex, Crossref, CIPO/IP Horizons CSV, fixtures.
2. **Normalization layer**: names, domains, institutions, Quebec filters.
3. **CVCD layer**: problem, method, object, result, evidence, limits, invariants, residue.
4. **OAK layer**: novelty, evidence, reproducibility, IP risk, feasibility, Tristan synergy.
5. **HGFM layer**: graph nodes/edges for documents, people, institutions, owners, domains.
6. **Bridge layer**: science↔patent candidate links and prior-art/transfer hypotheses.
7. **OAK-IP layer**: classification and release gates before public disclosure.
8. **Product layer**: dashboard, SQLite, DCT++ cards, opportunity matrix, reusable blueprints.

## Productizable forms

- internal research assistant;
- Quebec university innovation atlas;
- patent landscape assistant;
- startup opportunity miner;
- government innovation/public-service improvement radar;
- corporate Jarvis science-IP brain;
- GitHub Action for periodic digest snapshots.

## Non-negotiable OAK boundaries

- No legal advice.
- No automatic public disclosure of patentable ideas.
- No use of copyrighted full text unless licensed or otherwise permitted.
- No claim that a patent proves physical implementation.
- No claim that a publication is reproducible without checking methods/data/code.
- No confusion between inventor, author, assignee, owner, licensee and institution.

## PLUS ULTRA loop

```text
Observe → Ingest → Normalize → LOG/CVCD → OAK → HGFM → Bridges → Opportunities → Release Gates → DCT++ → GitHub Issues/PRs → M⁺/M⁻
```

## Reuse-first principle

Every output must be reusable as one of:

- code module;
- data schema;
- database table;
- dashboard panel;
- DCT++ card;
- GitHub issue;
- OAK review item;
- invention disclosure draft;
- publication/prototype backlog item.
