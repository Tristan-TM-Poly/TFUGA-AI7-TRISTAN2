# Ω-AUTO²-OAK-FIXALL-T Batch Report 0.1.0

This release adds a local dry-run batch reporting layer on top of the FixAll decision validator.

## Included

- Batch report generator.
- Markdown and JSON outputs.
- Example decision fixtures.
- Fixture list.
- Human runbook.
- M+ and M− registries.
- PR-only CI.

## Not included

- Live GitHub API ingestion.
- Automatic merge execution.
- Draft-to-ready transitions.
- External action connectors.

The module is intentionally advisory and local-first.
