# Outreach Guardrails — Quebec Universities

## Purpose

Protect credibility, avoid spam, and keep every outreach OAK-safe.

## Rules before sending

- Verify the university and the current contact page.
- Prefer one relevant office/person over generic mass sending.
- Customize the first paragraph for the institution.
- Do not send more than one initial message per target office without a reason.
- Wait 7-10 days before one follow-up.
- Stop after one polite follow-up unless invited to continue.
- Track every response as an execution card.

## Claims to avoid

```text
validated industrial breakthrough
guaranteed ROI
guaranteed performance
universal solution
official partnership before agreement
university endorsement before written confirmation
```

## Safe claims

```text
prototype framework
exploratory collaboration
benchmark proposal
OAK-safe validation approach
risk-adjusted value analysis
research and patent mapping concept
```

## Response handling

| Response | Action |
|---|---|
| Interested | prepare custom one-pager + meeting decision card |
| Redirect | thank them and prepare message to new contact |
| No response | one follow-up after 7-10 days |
| Rejection | record M-minus reason and improve positioning |
| Request for details | send one-pager + relevant GitHub/docs excerpt |

## Confirmation rule

```text
No meeting, partnership, endorsement, submission, or collaboration is considered confirmed without a saved confirmation record.
```
