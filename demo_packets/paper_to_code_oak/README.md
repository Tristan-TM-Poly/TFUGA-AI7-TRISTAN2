# Demo Packet Seed — Paper-to-Code OAK

## Purpose

Convert a scientific paper into a structured, OAK-safe artifact:

```text
paper → claims/equations/tables → code skeleton → baseline/reproduction plan → OAK report
```

## Status

```yaml
status: seed_packet
ready_to_send: false
next_action: build example using an open-access paper or user-provided paper
```

## Intended outputs

- paper summary;
- equations and assumptions;
- claim-evidence graph;
- code skeleton or notebook plan;
- reproduction constraints;
- OAK gaps and M− risks;
- university collaboration proposal.

## Best targets

- Université de Montréal / Mila-adjacent AI routes;
- McGill scientific AI / engineering;
- Concordia applied AI / engineering;
- UQAM data science / digital humanities;
- TÉLUQ learning systems;
- any lab asking for a paper digestion example.

## OAK lock

Do not present extracted equations/code as correct until source pages, licensing, assumptions, and reproduction checks are verified.
