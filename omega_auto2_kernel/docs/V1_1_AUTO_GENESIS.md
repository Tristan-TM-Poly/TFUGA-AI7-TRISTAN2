# Ω-AUTO-GENESIS v0.1 / AUTO² v1.1

## Goal

Transform an intention into a structured local draft report.

## Modules

- `genesis.py`
- `genesis_tree.py`
- `genesis_score.py`
- `genesis_report.py`

## CLI

```bash
auto2 genesis "create a revenue engine"
auto2 genesis "create a HEAL protocol" --mode max
auto2 genesis
```

## OAK

AUTO-GENESIS produces local draft text only. External effects remain outside this module.
