# Omni Translator V8.3

## Purpose

Project a factorized section object into multiple useful representations without confusing representation with proof.

## Layers

| Layer | Target | Output |
|---|---|---|
| L0 | machine | bytes, hashes, compact numeric arrays |
| L1 | orchestration | JSON contracts, SQL rows, Python/Rust/C++ stubs |
| L2 | expert human | LaTeX, IEEE-style prose, equations, benchmarks |
| L3 | public human | explanation, metaphor, course, atlas, fiction |

## Rule

```text
same vector -> multiple projections
projection != validation
```

## Required metadata per projection

- source section id,
- reality register R0-R5,
- EvidenceGate result,
- intended audience,
- blocked interpretations.

## Canon examples

- `v_n` -> JSON agent contract.
- `v_n` -> LaTeX equation stub.
- `v_n` -> README section.
- `v_n` -> public explanation.
- `v_n` -> fiction/manifest paragraph labeled R0.
