# GameMasterAcademy-T — OMEGA-GAME-T+++

`GameMasterAcademy-T` forme des GameMasters specialises pour piloter les mondes de Tristan : repo, code, energie, procedes, revenus, langues et OAK.

Il prolonge GameEngineOS-T :

```text
World -> Quest -> SkillTree -> Evaluation -> GameMasterProfile -> Better GameMaster
```

## Role

GameMasterAcademy transforme les competences en parcours jouables : observer, expliquer, simuler, tester, documenter, mesurer, corriger et garder les limites visibles.

Le systeme ne donne pas de certification officielle externe. Il produit une evaluation interne et des prochaines quetes d'entrainement.

## GameMasters MVP

- `RepoGM` : depots, docs, tests, schemas, review path ;
- `CodeGM` : lecture, invariants, tests, petits changements, docs ;
- `EnergyGM` : production, stockage, pertes, resilience, limites ;
- `ProcessGM` : procedes abstraits, qualite, circularite, prudence ;
- `RevenueGM` : hypotheses, offres, signaux, feedback, M+/M- ;
- `LanguageGM` : clarte, traduction d'intention, pedagogie, prudence.
