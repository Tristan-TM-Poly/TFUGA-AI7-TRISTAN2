# ProductBench-T — Ω-GAME-T+++

`ProductBench-T` mesure la qualité d'un produit issu de Ω-GAME-T.

OAKBench mesure le monde. ProductBench mesure le produit.

## Score

```text
ProductScore =
  0.18 value
+ 0.15 clarity
+ 0.15 differentiation
+ 0.15 feasibility
+ 0.15 testability
+ 0.12 revenue_potential
+ 0.10 strategic_fit
- 0.10 risk
- 0.08 scope_creep
- 0.07 ip_uncertainty
```

Le score est borné entre `0` et `1`.

## Seuils

| Score | Niveau |
|---|---|
| `>= 0.95` | plus ultra |
| `>= 0.90` | excellent |
| `>= 0.80` | priority |
| `>= 0.60` | prototype |
| `< 0.60` | needs work |

## Règle OAK

ProductBench est un signal de décision. Il n'est pas une preuve de revenu, de marché ou de validation scientifique.

- prioriser les produits ;
- détecter scope creep ;
