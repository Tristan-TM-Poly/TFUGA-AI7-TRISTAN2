# LanguageCurriculum-T

LanguageCurriculum-T turns PolyglotLanguageEngine-T and LanguageGM Rubric-T into a progressive training path.

## Purpose

It organizes language work into tracks, levels, quests, progress, M+/M-, and next training steps.

## Tracks

- `fr_clear`
- `en_clear`
- `teaching`
- `markdown_doc`
- `json_contract`
- `yaml_plan`
- `github_issue`
- `pitch`
- `ip_caution`

## Levels

```text
apprentice -> builder -> verifier -> strategist -> master
```

## Core loop

```text
CurriculumQuest -> LanguageQuest -> LanguageRun -> LanguageGMRubric -> CurriculumProgress -> next quest
```

## OAK boundary

LanguageCurriculum-T is an internal training system. It does not grant external credentials and does not replace review for sensitive public, business, policy, or claim-heavy writing.

## M+/M-

- M+ becomes skill reinforcement.
- M- becomes the next practice target.

## Reviewable split

This is a small split unit intended to stay reviewable and testable.
