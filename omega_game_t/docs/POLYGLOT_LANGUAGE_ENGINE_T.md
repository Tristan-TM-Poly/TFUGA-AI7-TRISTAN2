# PolyglotLanguageEngine-T

PolyglotLanguageEngine-T is a small OAK-safe split unit for Omega GAME T.

It turns human and technical language work into training quests for LanguageGM and related GameMasters.

## Purpose

The engine helps transform an idea into a clearer target form:

- plain French;
- plain English;
- teaching explanation;
- pitch draft;
- Markdown documentation;
- JSON contract;
- YAML plan;
- GitHub issue draft;
- IP caution note.

## Boundary

This is an internal simulation and drafting engine. It does not claim official translation, legal advice, patent advice, or external certification.

## Core loop

```text
LanguageQuest -> PolyglotLanguageEngine -> LanguageRun -> OAK notes -> M+/M-
```

## OAK rules

- keep uncertainty visible;
- preserve intent;
- adapt to audience;
- avoid false authority;
- avoid hidden claims;
- mark sensitive IP/legal/business content for review.

## GameMaster training use

LanguageGM learns to:

1. read rough ideas;
2. compress intent;
3. choose the right register;
4. produce a safe draft;
5. expose limits;
6. record M+ and M-;
7. propose the next language quest.
