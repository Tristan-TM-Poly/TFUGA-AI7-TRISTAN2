# Official Formation Checklist — Ω-CORP-JARVIS-T

This checklist prepares official company formation. It does not replace a lawyer, accountant, tax professional, bank, or government filing system.

## 0. Pre-legal preparation

- [ ] Choose initial architecture: one operating corporation first.
- [ ] Select 3 candidate names.
- [ ] Confirm jurisdiction strategy: Quebec vs federal plus Quebec registration.
- [ ] Fill founder legal name, address, email, phone.
- [ ] Fill registered office address.
- [ ] Confirm first director(s) and officer(s).
- [ ] Confirm initial shareholder and ownership split.
- [ ] Prepare share structure questions for lawyer.
- [ ] Prepare IP inventory and assignment/licensing questions.
- [ ] Prepare first offers and claims guardrails.

## 1. Professional review

- [ ] Lawyer/accountant reviews jurisdiction.
- [ ] Lawyer/accountant reviews share structure.
- [ ] Lawyer reviews IP assignment/licensing plan.
- [ ] Accountant reviews fiscal year and bookkeeping setup.
- [ ] Lawyer reviews first contract/NDA/proposal templates.
- [ ] Insurance needs reviewed for software/AI/consulting.

## 2. Official filing

Human action required.

- [ ] Search name availability.
- [ ] File incorporation/constitution.
- [ ] Pay government fees.
- [ ] Save articles/certificate/registration records.
- [ ] Obtain corporation number and/or NEQ where applicable.
- [ ] File initial declaration/registration if required.

## 3. Post-formation

- [ ] Create corporate minute book.
- [ ] Issue initial shares.
- [ ] Record director/officer appointments.
- [ ] Open corporate bank account.
- [ ] Create bookkeeping system.
- [ ] Register tax accounts if required.
- [ ] Execute IP assignment or license agreement.
- [ ] Create invoice template.
- [ ] Create contract template set.
- [ ] Create corporate email/domain if needed.

## 4. Operating launch

- [ ] Launch OAK Technical Audit offer.
- [ ] Launch ROI-OAK Commercialization Report offer.
- [ ] Prepare Datacenter Thermal OAK pilot deck.
- [ ] Create CRM/lead tracker.
- [ ] Create proposal tracker.
- [ ] Create OAK validation report template.
- [ ] Create monthly finance review.
- [ ] Create M-minus legal/commercial review.

## 5. Subsidiary triggers

Do not create subsidiaries until one or more triggers occur.

- [ ] Paid datacenter pilot.
- [ ] Signed LOI with enterprise partner.
- [ ] Patent filing or licensing portfolio need.
- [ ] Significant liability separation need.
- [ ] External investment round.
- [ ] Battery/BESS safety-reviewed partner project.
- [ ] Recurring SaaS/API revenue.

## OAK hold conditions

Stop and request professional review if any of these occur:

- [ ] investor solicitation
- [ ] loan or debt financing
- [ ] IP transfer to/from another person or entity
- [ ] contractor or employee hiring
- [ ] customer contract with liability exposure
- [ ] public performance or ROI claim
- [ ] government grant/subsidy application
- [ ] export-controlled or regulated domain
