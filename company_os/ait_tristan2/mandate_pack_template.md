# Mandate Pack Template — AIT-Tristan2

Internal template for assigning scoped work to a delegate or service. This is a preparation template and must be adapted/reviewed before real use.

## Delegate role

```text
Role: [admin / professional review / mail / banking preparation / product ops / other]
Name or service: [TODO]
Start date: [TODO]
End or review date: [TODO]
```

## Scope

```text
Allowed tasks:
- [TODO]
- [TODO]

Not allowed:
- actions outside written scope
- unapproved spending
- unapproved external statements
- unapproved submissions
- disclosure of confidential material
```

## Approval thresholds

```text
Needs approval before:
- external communication
- spending
- using identity documents
- public posting
- signing or submitting anything
- changing official records
```

## Confidentiality

```text
Confidential materials may only be used for the assigned task and must not be shared outside the approved scope.
```

## Proof required

```text
Every completed task must return one or more:
- completion note
- confirmation email
- screenshot
- PDF or document copy
- receipt
- timestamped checklist
```

## Reporting format

```yaml
delegate_report:
  task_id: TODO
  completed: true/false
  proof: []
  blockers: []
  next_action: TODO
```

## Termination / pause rule

```text
Pause immediately if scope, authority, cost, confidentiality, or safety becomes unclear.
```
