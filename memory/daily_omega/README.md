# Daily Ω Memory

This directory is reserved for M+ and M- updates produced by the Daily Ω Briefing / War Room system.

## Purpose

Daily intelligence should not disappear after being read. It should update memory:

```text
M+ = signals, tests, sources, and actions that proved fertile
M- = hype traps, weak sources, duplicate paths, unsafe paths, failed tests, and premature-publication risks
```

## Suggested files

```text
memory/daily_omega/m_plus.md
memory/daily_omega/m_minus.md
memory/daily_omega/source_quality_notes.md
memory/daily_omega/ip_disclosure_warnings.md
memory/daily_omega/canon_promotions.md
```

## Rule

A failed idea is not wasted if it becomes a reusable anti-error rule.
