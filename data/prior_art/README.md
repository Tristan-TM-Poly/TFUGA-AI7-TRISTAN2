Protect Tristan's invention pipeline by separating:

```text
open/public artifacts
```

## OAK rule

Prior art can inspire, limit, falsify, or redirect a project. It should never be ignored because it is inconvenient.

## Disclosure rule

If a Daily Ω item creates a potentially patentable Tristan-generated direction, record it privately first and avoid public disclosure until the IP posture is clarified.
